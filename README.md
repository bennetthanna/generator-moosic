# Moosic Yeoman Generator

Interactive CLI tool to upload file(s) to S3

## Installation
1. Install repo
```
git clone https://github.com/bennetthanna/generator-moosic.git
```

2. Install packages
```
npm install
```

3. Link package
```
npm link
```

4. Follow the prompts and upload that moosic!
```
yo moosic
```
